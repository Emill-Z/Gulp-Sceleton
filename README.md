# Gulp Skeleton
### Installation
What you need:
1. [NPM](https://www.npmjs.com/)
2. [Gulp v4](http://gulpjs.com/)

#### Then install all dependencies
```bash
npm install
```

##### For dev
Execute `npm start` (This will run: build, watch and browser sync)

##### For build prod
Execute `npm run build` (This will minify: images, js, css)
